# LyricDock v1.0.0

## 亮点

- 菜单栏实时显示封面、歌名歌手和歌词
- 支持 `上一首 / 播放暂停 / 下一首`
- 支持 `Apple Music / Spotify / 汽水音乐 / 网易云音乐 / QQ 音乐`
- 过滤网页等非目标播放源，避免异常接管显示

## 下载

- `LyricDock.app`
- `LyricDock-<version>-<build>.zip`

## 首次使用

1. 打开应用后，允许播放器自动化权限
2. 如果使用 Widget，请确认 App Group 已配置
3. 如遇系统安全提示，可在系统设置里手动允许应用运行

## 已知说明

- 当前版本更适合独立分发，不建议按 Mac App Store 路线理解
- 歌词依赖外部歌词源，少量歌曲可能没有歌词或时间轴不完整
- 某些播放器能力取决于它们是否正确公开系统媒体会话
